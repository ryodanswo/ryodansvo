# Складской учет

## Краткое описание

Приложение предназначено для автоматизации учета складских товаров. Оно позволяет эффективно управлять запасами продукции, отслеживать количество товаров, категории, статусы и сроки годности. Приложение ориентировано на сотрудников склада и менеджеров, занимающихся логистикой и управлением товарными остатками.


## Особенности

* Добавление новых товаров и категорий.
* Управление поставщиками и движением товаров (приходы и расходы).
* Просмотр статистики запасов и остатков товаров.
* Уведомления о приближающемся сроке годности товаров.

## Установка

Чтобы установить проект, выполните следующие шаги:

1.  Клонируйте репозиторий:
    ```
    git clone https://github.com/yourusername/scm-project.git
    ```
2.  Перейдите в директорию проекта:
    ```
    cd scm-project
    ```
3.  Создайте базу данных PostgreSQL и настройте подключение в файле:
    ```
    application.properties
    ```
4.  Запустите приложение:
    ```
    ./mvnw spring-boot:run
    ```
