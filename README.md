# Складской учет
# Описание проекта
Приложение предназначено для автоматизации учета складских товаров. Оно позволяет эффективно управлять запасами продукции, отслеживать количество товаров, категории, статусы и сроки годности. Приложение ориентировано на сотрудников склада и менеджеров, занимающихся логистикой и управлением товарными остатками.

Основные функции приложения включают:

Добавление новых товаров и категорий.
Управление поставщиками и движением товаров (приходы и расходы).
Просмотр статистики запасов и остатков товаров.
Уведомления о приближающемся сроке годности товаров.
Установка и запуск
Предварительные условия
Для запуска приложения необходимы следующие компоненты:

JDK 11+
Maven 3.x
База данных PostgreSQL
### Шаги установки
1.Клонируйте репозиторий:git clone https://github.com/ryodansvo/scm-project.git
2.Перейдите в папку проекта:cd scm-project
3.Создайте базу данных PostgreSQL и настройте подключение в файле application.properties.
4.Запустите приложение:./mvnw spring-boot:run
или соберите исполняемый JAR-файл:./mvnw clean packagejava -jar target/scm-project.jar
